個人ブログ。Next.js 15 App Router + TypeScript + Contentlayerで、`data/posts/**/*.md` と `data/authors/**/*.md` を処理して静的サイトとして出力する。`next.config.js` は `output: "export"`。記事データは `contentlayer/generated` の `allPosts` / `allAuthors` から参照し、タグ集計は `contentlayer.config.ts` の `onSuccess` で `src/tagList.json` を生成する。主要ディレクトリは `src/app`（ルート/ページ）、`src/components`（レイアウト/UI）、`src/features/posts` と `src/features/tags`（機能別）、`data`（コンテンツ）、`public`（静的資産）。
